# Student Grade Calculator

## Project Description
A comprehensive grade calculator that processes multiple students' marks, calculates grades with personalized comments, and provides class statistics.

## What I Learned
1. **Condtional logic**: Using if/elif/else for decision making
2. **Lists**: Storing and manipulating collections of data
3. **Loops**: Using for and why loops for repetition
4. **Error handling**: Using try-except to handle invalid inputs
5. **Functions**: Organizing code into reusable blocks

## Fetures
- Processes multiple students
- Calculates grades based on custon grading system
- Provides personalized comments for students
- Calculates class statistics
- Formatted table output
- Input validation for all users inputs
- Error handling for edge cases

## Grading System
+ A: 90-100 (Excellent)
+ B: 80-89 (Very good)
+ C: 70-79 (Room for improvement)
+ D: 60-69 (Needs improvement)
+ F: 0-59 (Failed)

## How to Run
1. Navigate to project folder
2. Run the following command in terminal/command prompt: `python grade_calculator.py`
