# Personal Finance Tracker

## Description
A comprehensive personal finance tracking application that helps user manage their expenses, categorize spendings, and generate insightful reports.

## Features
+ Manage expenses using the comprehensive menu
+ Categorize expenses (food, bills, etc.)
+ Save data to JSON file with automatic saving
+ Load data on startup while checking for errors
+ Generate trend graphs and reports
+ view all expenses
+ Set budget limits and track them
+ Export to CSV available
+ User-friendly menu

## How to Run
- Navigate to project folder
- Launch terminal or CMD and enter the following `python main.py`
- Your're ready to track your expenses now!
